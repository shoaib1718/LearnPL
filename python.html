Introduction

History of Python.

Python is dynamically typed and garbage-collected. It supports multiple programming paradigms, including structured (particularly, procedural), object-oriented, and functional programming. It is often described as a "batteries included" language due to its comprehensive standard library.
Python was conceived in the late 1980s by Guido van Rossum at the Centrum Wiskunde & Informatica (CWI) in the Netherlands as a successor to the ABC programming language. Python 2.0 was released in 2000 and Python 3.0 in 2008. Python 2.0 is no longer supported and Python 3.0 is the current major version.
Python is one of the most popular programming languages in the world. It is used for a wide variety of applications, including web development, software development, task automation, data analysis, and data visualization. Python is also a popular language for teaching programming.

Variables

A variable in Python is a container for storing data values. Unlike other languages, such as Java, Python has no command for declaring a variable, so you create one the moment you first assign a value to it. For example, the following code creates a variable called name and assigns it the value "Alice":

name = "Alice"

You can then use the variable name in your code, as in the following example:

print(name)

This code will print the value of the variable name, which is "Alice". Variables can be of any data type, such as integers, floats, strings, or booleans. You can also create lists, tuples, dictionaries, and sets, which are all collections of data values. Here are some examples of variables of different data types:

# Integer
age = 25

# Float
height = 1.75

# String
name = "Alice"

# Boolean
is_adult = True

# List
friends = ["Bob", "Carol", "Dave"]

# Tuple
coordinates = (10, 20)

# Dictionary
phone_book = {"Alice": 1234567890, "Bob": 9876543210}

# Set
unique_numbers = {1, 2, 3, 4, 5}

You can use variables in your code to perform calculations, store data, and control the flow of your program.
